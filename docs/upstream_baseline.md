# Upstream Baseline

This repository is derived from the public upstream REFLECT repository:

- Upstream repo: <https://github.com/real-stanford/reflect> (MIT License)
- Project page: <https://robot-reflect.github.io/>
- Dataset: <https://www.cs.columbia.edu/~liuzeyi/reflect_data>
- Paper: Liu, Bahety, Song - *REFLECT: Summarizing Robot Experiences for Failure
  Explanation and Correction* (arXiv:2306.15724, 2023)
- Upstream focus: simulation-first failure explanation and correction using
  hierarchical summaries plus LLM reasoning

Everything this fork adds or changes relative to upstream is registered in
[fork_changes.md](fork_changes.md).
